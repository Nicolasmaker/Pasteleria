# Pastelería Los Sabores

Proyecto de pasteleria, para prueba final de full stack.
Desarrollado con Node.js, TypeScript y Vite.

## Requisitos

- Node.js instalado.

## Instalación

```bash
npm install
```

## Desarrollo

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

## Construcción

Para compilar el proyecto para producción:

```bash
npm run build
```
