# Pastelería Los Sabores

Proyecto web para la Pastelería Los Sabores, desarrollado con Node.js, TypeScript y Vite.

## Requisitos

- Node.js instalado.

## Instalación

```bash
npm install
```

## Desarrollo

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

## Construcción

Para compilar el proyecto para producción:

```bash
npm run build
```
